# E-Banking Application
#### Cette une application qui gérer les clients et leurs comptes avec l'historique de l'ensemble des opération effectées. aussi la possibilité de créditer, débuter dans un compte et le transfert d'argent entre deux comptes.
#### L'application est sécurisée par une authentification JWT.
